# AI Investment Research Agent

> Full-stack AI-powered investment research tool built with Next.js, LangGraph, and Gemini.

**Status:** Project scaffold — agent logic not yet implemented.

## Quick Start

```bash
npm install
cp .env.local.example .env.local   # then fill in your API keys
npm run dev
```

## Tech Stack

- **Framework:** Next.js 14+ (App Router, TypeScript)
- **Styling:** Tailwind CSS
- **AI Orchestration:** LangGraph.js + LangChain.js
- **LLM:** Google Gemini (`@google/generative-ai`)
- **Search:** Tavily API
- **Financial Data:** Alpha Vantage API
